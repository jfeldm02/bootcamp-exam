The Network as an Equation
A neural network is, at its core, a stacked composition of "weighted sum + non-linear bend":
ŷ = softmax( W₂ · σ( W₁ · x + b₁ ) + b₂ )
Each piece, mapped back to the bootcamp:
W₁·x + b₁ — matrix-vector multiply (Lec 2). It's still ax + b = 0, just in higher dimensions.
σ(·) — the activation function bends the line into a curve.
softmax(·) — turns scores into probabilities (Lec 4).
θ ← θ − α·∇ℒ — gradient descent updates every weight (Lec 3).
train/test split — estimating generalization (Lec 5).
