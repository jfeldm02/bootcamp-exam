# Justin's Math in AI Bootcamp

A self-contained final exam and ML visualization demo covering the four mathematical foundations of machine learning: Linear Algebra, Calculus, Probability, and Statistics.

## Files

| File | Description |
|------|-------------|
| `bootcamp_final_exam_official.html` | 12-question final exam with instant feedback, scoring, and a downloadable certificate of completion |
| `ml_visualizer.html` | Interactive neural network visualization demo — unlocked on the exam results page after a perfect score |

## Live Links

- **Exam:** https://jfeldm02.github.io/bootcamp-exam/bootcamp_final_exam_official.html
- **ML Demo:** https://jfeldm02.github.io/bootcamp-exam/ml_visualizer.html — Password: `CalvinandHobbes`

## Topics Covered

- **Linear Algebra** — dot products, matrix multiplication, cosine similarity
- **Calculus** — derivatives, partial derivatives, gradient descent
- **Probability** — independent events, Bayes' theorem, conditional probability
- **Statistics** — mean, median, mode, standard deviation, z-scores
